# Tableau Executive Dashboard & Data Storytelling

## Business Problem Summary
This project builds an executive Tableau dashboard to help retail leadership track sales performance, profitability, customer behaviour, regional results, shipping efficiency, discount impact, and product returns. The dashboard is designed to enable data-driven decision-making through interactive visualisations.

---

## Dataset Description
The dataset contains retail order transactions covering customer, product, sales, profit, shipping, and return information.

Key fields include:

* Order Date
* Region
* Customer Segment
* Category
* Sub-Category
* Ship Mode
* Sales
* Profit
* Discount
* Delivery Days
* Return Flag
* Campaign Channel

---

## Tableau Workbook Description
The workbook contains interactive worksheets and a single executive dashboard that consolidates key business performance metrics.

Views included:

* Sales Trend & Growth Analysis
* Regional Performance Dashboard
* Product Category Profitability
* Customer Segment Analysis
* Shipping & Delivery Performance
* Impact of Discount on Profitability
* Return Pattern Analysis

---

## Calculated Fields Created

* Profit Margin
* Cost
* Average Order Value
* Return Rate
* Shipping Delay Bucket

---

## Dashboard Components
The dashboard is made up of the following components:

* KPI Cards
* Sales Trend
* Regional Performance
* Category Profitability
* Discount vs Profit
* Return Analysis

---

## Filters and Interactions Used
Interactive filters available to users include:

* Region
* Category
* Customer Segment
* Ship Mode

Dashboard action filters allow users to select a data point in one chart and have all remaining visualisations update dynamically in response.

---

## Key Business Insights

* Sales levels fluctuate across different periods of the year.
* The South region produced the highest overall sales figures.
* Technology products delivered the strongest profit performance.
* Customer segments contribute broadly similar levels of total sales.
* Increased discounting is generally associated with reduced profitability.
* Standard Class shipments experience the longest average delivery times.
* Furniture records the greatest volume of product returns.
* Interactive filtering supports detailed regional performance exploration.

---

## Dashboard Story Summary
The dashboard surfaces trends in revenue growth, regional strengths, high-performing product categories, customer purchasing behaviour, shipping efficiency, discount effectiveness, and return risks. It gives leadership the tools to identify revenue growth opportunities while keeping a close eye on operational risks.

---

## Assumptions and Limitations

* Calculated fields were derived using the fields available within the provided dataset.
* The dashboard reflects only the data included in the source dataset.
* Return Rate is calculated based on the return records available in the dataset.
* Business conclusions may shift as additional historical data becomes available.
